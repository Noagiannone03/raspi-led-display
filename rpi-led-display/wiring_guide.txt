⚠️  IMPORTANT - ALIMENTATION ⚠️
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
NE CONNECTEZ PAS l'alimentation 5V de l'écran au Raspberry Pi !

• Utilisez une alimentation externe 5V/4A minimum (6A recommandé)
• Reliez UNIQUEMENT les pins GND ensemble
• Un écran 64x64 peut consommer jusqu'à 4A à pleine luminosité

UTILISATION D'UN HAT ADAFRUIT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Si vous utilisez un Adafruit RGB Matrix Bonnet ou HAT :

1. Empilez le HAT sur le Raspberry Pi
2. Connectez le câble ruban du LED panel au HAT
3. ⚠️  SOUDEZ le jumper E sur le HAT (obligatoire pour 64x64)
4. Alimentation externe 5V/4A sur le bornier du HAT
5. Modifiez le code : hardware_mapping = 'adafruit-hat'


VÉRIFICATION DU CÂBLAGE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ Checklist avant de démarrer :

□ Tous les câbles de données (R1, G1, B1, R2, G2, B2) sont connectés
□ Toutes les lignes d'adresse (A, B, C, D, E) sont connectées
□ Les signaux de contrôle (CLK, LAT, OE) sont connectés
□ Le GND est connecté entre le Pi et l'écran
□ L'alimentation 5V externe est branchée SUR L'ÉCRAN (pas sur le Pi)
□ Les connexions sont bien enfoncées et sécurisées


PROBLÈMES COURANTS
